Valentino's Anime Gear - demo site (90s-inspired)

Folder structure:
- index.html
- styles.css
- script.js
- images/
  - slayers_shirt_1.jpg
  - bubblegum_hoodie_1.jpg
  - gunsmith_pants_1.jpg
  - shadowskill_jacket_1.jpg
  - dominion_shirt_1.jpg
  - akira_hoodie_1.jpg
  - ninja_pants_1.jpg
  - clamp_sweater_1.jpg

Quick start:
1. Place the files in a folder named valentinos-anime-gear and add the images listed above into the images/ folder.
2. Open index.html locally to test. For better local testing use a static server:
   - python -m http.server 8000
   - open http://localhost:8000
3. To publish: push the folder to GitHub (repo name valentinos-anime-gear) and enable GitHub Pages (Settings → Pages → branch main → root).
4. Replace placeholder images with your own licensed/original artwork before publishing publicly.

Notes:
- This demo intentionally uses placeholder filenames for training. Do not upload copyrighted character art without permission.
